PTAP (对等可信应用扩展协议)

  PTAP（Peer Trusted Application Protocols）是基于PPk技术社区所定义的ODIN命名标识和PTTP传输协议，为具体应用而定义的扩展协议，带有具体编号（如PTAP02）。 每一个扩展协议描述了内容提供节点即AP(Access Point)按照PTTP协议提供特定应用内容时，对应的资源标识（URI）和应答内容相关的标准定义。

  PTAP are extended protocols defined for specific applications based on the ODIN naming and PTTP transport protocol.Each PTAP protocol has a specific number (eg PTAP02). Each PTAP protocol describes the standard definition of requested resource identifier (the URI scheme based ODIN uased as PTTP request) and responsed data (the content of PTTP data package) , when the AP(Access Point,the service provider node) provides a specific application content according to the PTTP protocol.
